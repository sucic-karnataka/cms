Hugo ಜಗತ್ತಿನ ಅತ್ಯಂತ ವೇಗವಾದ ಸ್ಥಿರ ಸೈಟ್ ಜೆನರೇಟರ್‌ಗಳಲ್ಲಿ ಒಂದಾಗಿದೆ. Go ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಭಾಷೆಯಲ್ಲಿ ನಿರ್ಮಿಸಲಾದ ಈ ಉಪಕರಣ ಸಾವಿರಾರು ಪುಟಗಳನ್ನು ಕೆಲವೇ ಸೆಕೆಂಡುಗಳಲ್ಲಿ ನಿರ್ಮಿಸಬಲ್ಲದು.

<!--more-->

## Hugo ಯಾಕೆ ಬಳಸಬೇಕು?

Hugo ಗೆ ಹಲವಾರು ಪ್ರಯೋಜನಗಳಿವೆ:

- **ವೇಗ**: ಸಾವಿರ ಪುಟಗಳ ಸೈಟ್ ಅನ್ನು ಒಂದು ಸೆಕೆಂಡಿಗಿಂತ ಕಡಿಮೆ ಸಮಯದಲ್ಲಿ ನಿರ್ಮಿಸುತ್ತದೆ
- **ಸರಳತೆ**: ಯಾವುದೇ ಡೇಟಾಬೇಸ್ ಅಗತ್ಯವಿಲ್ಲ
- **ಮುಕ್ತ ಮೂಲ**: ಉಚಿತ ಮತ್ತು ಮುಕ್ತ ಮೂಲ ತಂತ್ರಾಂಶ
- **Markdown ಬೆಂಬಲ**: ಸರಳ Markdown ಫಾರ್ಮ್ಯಾಟ್‌ನಲ್ಲಿ ವಿಷಯ ಬರೆಯಿರಿ
- **ಬಹುಭಾಷಾ ಬೆಂಬಲ**: ಕನ್ನಡ ಸೇರಿದಂತೆ ಅನೇಕ ಭಾಷೆಗಳಲ್ಲಿ ಸೈಟ್ ರಚಿಸಿ

## Hugo ಸ್ಥಾಪನೆ

macOS ನಲ್ಲಿ Homebrew ಮೂಲಕ:

```bash
brew install hugo
```

Ubuntu/Debian ನಲ್ಲಿ:

```bash
sudo apt-get install hugo
```

Windows ನಲ್ಲಿ Chocolatey ಮೂಲಕ:

```bash
choco install hugo
```

## ಮೊದಲ Hugo ಸೈಟ್ ರಚಿಸಿ

```bash
hugo new site ನನ್ನ-ಬ್ಲಾಗ್
cd ನನ್ನ-ಬ್ಲಾಗ್
```

ಒಂದು ಥೀಮ್ ಸೇರಿಸಿ:

```bash
git init
git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke themes/ananke
echo "theme = 'ananke'" >> hugo.toml
```

ಅಭಿವೃದ್ಧಿ ಸರ್ವರ್ ಆರಂಭಿಸಿ:

```bash
hugo server -D
```

## Hugo ನ ಫೋಲ್ಡರ್ ರಚನೆ

Hugo ಸೈಟ್‌ನಲ್ಲಿ ಈ ಕೆಳಗಿನ ಪ್ರಮುಖ ಫೋಲ್ಡರ್‌ಗಳಿವೆ:

| ಫೋಲ್ಡರ್ | ಉದ್ದೇಶ |
|---------|--------|
| `content/` | ಎಲ್ಲ ವಿಷಯ ಫೈಲ್‌ಗಳು |
| `layouts/` | HTML ಟೆಂಪ್ಲೇಟ್‌ಗಳು |
| `static/` | ಚಿತ್ರಗಳು, CSS, JS |
| `themes/` | ಡಿಸೈನ್ ಥೀಮ್‌ಗಳು |
| `public/` | ನಿರ್ಮಿತ ಸೈಟ್ ಫೈಲ್‌ಗಳು |

## ಮೊದಲ ಲೇಖನ ಬರೆಯಿರಿ

```bash
hugo new articles/ನನ್ನ-ಮೊದಲ-ಲೇಖನ/index.md
```

ಇದು `content/articles/ನನ್ನ-ಮೊದಲ-ಲೇಖನ/index.md` ಫೈಲ್ ರಚಿಸುತ್ತದೆ. ಅಲ್ಲಿ ನಿಮ್ಮ ವಿಷಯ ಬರೆಯಿರಿ.

## ಸೈಟ್ ನಿರ್ಮಿಸಿ

```bash
hugo
```

ಈ ಆದೇಶ `public/` ಫೋಲ್ಡರ್‌ನಲ್ಲಿ ಸಂಪೂರ್ಣ ಸ್ಥಿರ ಸೈಟ್ ನಿರ್ಮಿಸುತ್ತದೆ. ಈ ಫೈಲ್‌ಗಳನ್ನು Netlify, GitHub Pages ಅಥವಾ ಯಾವುದೇ ವೆಬ್ ಸರ್ವರ್‌ನಲ್ಲಿ ನಿಯೋಜಿಸಬಹುದು.

## ತೀರ್ಮಾನ

Hugo ಬಳಸಿ ವೇಗವಾದ, ಸುರಕ್ಷಿತ ಮತ್ತು ಕಡಿಮೆ ನಿರ್ವಹಣಾ ವೆಚ್ಚದ ವೆಬ್‌ಸೈಟ್ ರಚಿಸಿ. ಕನ್ನಡ ಬ್ಲಾಗ್ ಅಥವಾ ಸುದ್ದಿ ಸೈಟ್ ಆರಂಭಿಸಲು Hugo ಅತ್ಯುತ್ತಮ ಆಯ್ಕೆ.
